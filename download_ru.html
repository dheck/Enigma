Получить Enigma

Все ссылки и некоторые советы по загрузке Enigma представлены на этой странице и отсортированы по следующим критериям:
  • Стабильный выпуск - Самые новые выпущенные пакеты - начните отсюда.
  • Сборка из исходных кодов - Советы для Unix.
  • Список рассылки с уведомлением о выпуске - Получать по почте уведомление о выходе нового выпуска Enigma.
  • Разрабатываемые версии - Скачать самые последние исходные коды.
  • Старые версии

Стабильный выпуск (Версия 1.01)

Наряду с 'Полными' пакетами, мы предлагаем урезанные 'Минимальные' пакеты для пользователей с узкой полосой пропускания. Пакеты 'Минимальный' и 'Исходные коды' не включают в себя пакет 'Эскизы'. Пакеты при этом абсолютно полные, потому что не включенные эскизы сгенерируются на лету в течение нескольких секунд при первом вызове меню уровней.

Числа md5 и sha256 ниже это контрольные суммы, с помощью которых Вы можете проверить подлинность и полноту Ваших загрузок. Они не влияют на установку и процесс игры в Enigma.

    • sha256 - 61f86ff176865ca0cfa609c42102d5cb8f5318a58c2589b7bcf6a94f9d12478f
    • Windows '95 и выше Минимальный мастер установки (1.01) прибл. 11 MB
    • md5 - 831bfacc65ccbd797b22875a03117422
    • sha256 - 9613302c086f1f380d6f8a4cfd090a5f2527ebf4eeb5a894e854cd1d074462cd
    • Windows '95 и выше Минимальный Zip-архив (1.01) прибл. 11 MB
    • md5 - 59d670b828186e0bbe03626f3d31ab59
    • sha256 - 738f0f3f0827555543d0e33f43b3308da9b751d9504011eaeca1614bc38a891e
    • Универсальный мастер установки для Mac OS X Полный мастер установки (1.01) прибл. 18 MB
    • md5 - 693576941ab19020d82b44095e0f4699

Все пакеты доступны на Berlios. В случае возникновения проблем с загрузкой попробуйте скачать напрямую с Berlios попытаться скачать со всех предложенных серверов.

Пожалуйста сообщите о проблемах с любым из этих пакетов в список рассылки Enigma.

Пакеты для некоторых других операционных систем предоставляются третьими лицами:

  • Debian
  • FreeBSD  (версия 0.92)
  • Gentoo Linux
  • Ubuntu
  • GP2X  (версия 0.92)

Сборка из исходных кодов под Unix

Если Вы не нашли пакетов Enigma для своего дистрибутива Unix, Вы можете попытаться скачать пакет с исходными кодами и собрать Enigma самостоятельно. Обычно это самый короткий путь к цели.

Enigma зависит от нескольких библиотек и инструментов. Большинство Unix-систем поставляются с уже установленными необходимыми библиотеками. Но для сборки, помимо пакетов с выполняемыми библиотеками, Вам нужны дополнительные пакеты '-devel'. Пожалуйста проверьте, что в Вашей системе установлены библиотеки SDL, SDL_ttf, SDL_image, SDL_mixer и Xerces.

За полным списком библиотек и программ, которые должны быть установлены в Вашей системе для сборки Enigma, пожалуйста обратитесь к файлу 'doc/REQUIREMENTS', который включен в пакет исходных кодов Enigma.

Как только все будет установлено, создание Enigma станет простым как набор команд

./configure && make && make install

в директории enigma.

Если в Вашем дистрибутиве нет необходимых пакетов, Вы можете скачать их со следующих веб-сайтов:

  • SDL  (≥ 1.2)
  • SDL_ttf  (≥ 2.0.6)
  • SDL_mixer  (≥ 1.2.4)
  • SDL_image  (≥ 1.2.0)
  • Xerces  (≥ 2.4)

Список рассылки с уведомлением о выпуске

Если Вы задались целью получить самую последнюю версию Enigma, мы предлагаем услугу уведомления о новых выпусках по электронной почте. Просто зарегистрируйтесь в списке рассылки с уведомлением о выпуске. Объем почтового трафика будет ограничен одним сообщением о бета или финальном выпуске Enigma, который еще не превосходил полудюжины сообщений в течение года.

Разрабатываемые версии

Заинтересованы в самой последней разрабатываемой версии Enigma? Разрабатываемые версии Enigma можно скачать с репозитория их исходных кодов.

svn://svn.berlios.de/enigma-game

За подробными инструкциями обратитесь пожалуйста к странице разработки.

Старые версии

Из-за изменений в репозитории исходных кодов и серверах загрузки, старые версии Enigma можно найти на различных серверах. Посмотрите на

  • Berlios  если ищете недавние выпуски.
  • http://savannah.nongnu.org/download/enigma/  если ищете предыдущие выпуски
  • mag-heut.net  если ищете старые Windows-версии.

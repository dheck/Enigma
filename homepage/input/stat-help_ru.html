Пояснения

всего

Все распространяемые уровни подсчитываются. Если для уровня доступен упрощенный режим, обе версии принимаются в расчет. Уровни, которые появляются несколько раз в разных пакетах уровней просто считаются единожды.

совместных

Результат измеряется с разрешением в 1 секунду. Если два или больше пользователя завершили уровень с точностью до секунды у них у обоих будет тот же результат и они разделят между собой мировой рекорд. Мировой рекорд засчитывается как мировой рекорд для обоих конкурентов.

гандикап (hcp)

Для игроков, которые не заостряются на мировых рекордах, но все же стремятся найти прямое и быстрое решение уровней, мы ввели другую меру. Мы назвали ее гандикап из-за схожести с гандикапом в гольфе. В общем меньшее значение лучше, чем большее. Диапазон колеблется от +100 до -300. Игрок, который решает уровни за среднее время получает гандикап 0.0. Гандикап представляет сумму результатов решений игроков из средних профессиональных результатов (PAR), которые имеются для каждого уровня. Чтобы сделать значения гандикапа сопоставимыми они всегда нормализуются к 100 уровням - стандартный размер пакета уровней.

решенный гандикап

Решенный гандикап это гандикап, который рассчитывается для всех уровней, которые решил игрок. Дополнительно 10% нерешенных уровней берутся в расчет в случае неудавшейся попытке решения. Этот способ решенного гандикапа может использоваться для сравнения игроков с весьма отличающимися результатами решения. Он показывает насколько быстро или медленно игрок решает уровень.
